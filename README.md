# Basic RSA application
The goal of this project is to develop a straightforward application that securely encrypts and decrypts private communications using the RSA method. Additionally, it can produce a digital signature to confirm the authenticity and integrityÂ of the data and to prevent the signer from rejecting the contribution of others.
